---
title: "Get Tickets by User"
api: "GET /api/tickets/usuario/{usuarioId}"
description: "Retrieve all tickets created by a specific user"
---

## Authentication

This endpoint requires JWT authentication. Include the bearer token in the Authorization header.

<Note>
  Authentication is currently commented out in the controller but should be enabled in production.
</Note>

## Path Parameters

<ParamField path="usuarioId" type="string" required>
  The UUID of the user whose tickets you want to retrieve
</ParamField>

## Response

Returns an array of ticket objects created by the specified user.

<ResponseField name="tickets" type="array">
  Array of ticket objects
  
  <Expandable title="Ticket Object">
    <ResponseField name="IdTicket" type="string">
      Unique identifier (GUID) for the ticket
    </ResponseField>
    
    <ResponseField name="Codigo" type="string">
      Tracking code for the ticket
    </ResponseField>
    
    <ResponseField name="Titulo" type="string">
      Title of the ticket
    </ResponseField>
    
    <ResponseField name="Descripcion" type="string">
      Detailed description of the ticket
    </ResponseField>
    
    <ResponseField name="Area" type="string">
      Name of the area/department
    </ResponseField>
    
    <ResponseField name="Prioridad" type="string">
      Priority level name
    </ResponseField>
    
    <ResponseField name="Estado" type="string">
      Current status name
    </ResponseField>
    
    <ResponseField name="FechaCreacion" type="string">
      Timestamp when the ticket was created (ISO 8601 format)
    </ResponseField>
    
    <ResponseField name="FechaActualizacion" type="string">
      Timestamp when the ticket was last updated (ISO 8601 format)
    </ResponseField>
  </Expandable>
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X GET https://api.example.com/api/tickets/usuario/3fa85f64-5717-4562-b3fc-2c963f66afa6 \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

```csharp C#
var client = new HttpClient();
client.DefaultRequestHeaders.Authorization = 
    new AuthenticationHeaderValue("Bearer", "YOUR_JWT_TOKEN");

var usuarioId = Guid.Parse("3fa85f64-5717-4562-b3fc-2c963f66afa6");
var response = await client.GetAsync(
    $"https://api.example.com/api/tickets/usuario/{usuarioId}"
);

var tickets = await response.Content.ReadFromJsonAsync<List<ConsultarTicketDto>>();
```

</CodeGroup>
